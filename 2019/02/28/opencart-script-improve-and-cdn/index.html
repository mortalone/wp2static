Opencart script improve and cdn

    Posted by admin February 28, 2019 in Uncategorized

    We are working on new modules that put javascript and stylesheet files on powerful CDN networks.

    there wil be a paid pro version wich also combines all the built in javascripts and stylesheets into one file. and delivers them via jsdelivr

    jsDelivr – A free, fast, and reliable Open Source CDN for npm and GitHub
    A free, fast, and reliable Open Source CDN for npm and GitHub with the largest network and best performance, perfectly suited for production use.

    On the opencart default/stock frontpage 12 external javascript and stylesheets are loaded.

    with the pro module. this is cut down to 3 external files. wich are located on the very powerful jsdelivr network.

    With the pro version all new extensions that are not part of the opencart core system wil be served via cloudinary CDN server network. but they wil not be combined until such a service is present
