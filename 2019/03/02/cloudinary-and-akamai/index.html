The power of Cloudinary and Akamai

    Posted by admin March 2, 2019 in Uncategorized

    Multi-CDN switching

    So. Cloudinary as such is a image processing service. and it does this service to perfection, with a great number of options to enhance, improve or manipulate images. Most options are on the fly. with url or API calls.

    but on top of that Cloudflare uses the entire network of Akamai.


    The Akamai Intelligent Platform is a global cloud network for accelerating and securing web content and web application delivery. The Akamai cloud architecture spans more than 160,000 servers in over 95 countries. Over 90% of Internet users are a single network hop away from Akamai’s cloud servers, and at any given time Akamai delivers from 15-30% of total global Internet traffic.

    https://www.akamai.com/uk/en/resources/cloud-network.jsp

    The world’s largest content distribution network, owned and operated by Akamai, spans more than 216,000 servers in over 120 countries and within more than 1,500 networks around the world.

    https://www.akamai.com/uk/en/resources/content-distribution-network.jsp

    The two servies together offers a very powerfull tool for delivering static files as fast as possible to users all over the world. called Dynamic multi-CDN switching. wich you can read much more about in the link below

    New Multi-CDN solution auto-selects best CDN for optimal per-user experience
